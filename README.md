# Real-Time Monitoring Framework and Alerting System for Underground Coal Mine Using Arduino Uno

## Overview

This project is designed to improve coal mine safety by continuously monitoring hazardous environmental conditions such as gas leakage, temperature rise, and fire accidents using Arduino Uno.

## Components Used

* Arduino Uno
* MQ-2 Gas Sensor
* LM35 Temperature Sensor
* IR Flame Sensor
* HC-05 Bluetooth Module
* LCD Display
* Buzzer

## Features

* Real-time gas leak detection
* Temperature monitoring
* Fire detection system
* Instant buzzer alerts
* LCD display status monitoring
* Bluetooth-based remote monitoring

## Working Principle

The system continuously reads sensor data from MQ-2, LM35, and Flame Sensor. If any sensor value crosses the threshold, the Arduino triggers the buzzer and displays warning messages on the LCD. Sensor values are also transmitted through Bluetooth for remote monitoring.
